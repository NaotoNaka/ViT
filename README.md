# ViT
a simple ViT and CNN pytorch implementation

## Requirement
```
pip install -r requirements.txt
```

## Usage
For ViT training/testing:
```
python wrapper.py 0
```

For CNN training/testing:
```
python wrapper.py 1
```

## Reference
